<b>Graph Drawing</b><br>
This is a simple project about graphs and SVG drawing written in C++<br><br>
<b>INPUT</b><br>
The input is read from "graf.in" in the form of node,label and then the arcs between the nodes.<br>
<br>
<b>OUTPUT</b><br>
Afterwards you have the choice to draw your Graph in 2 modes: Circle and Random layout and a image is generated where nodes,labels and arcs are drawed accordingly<br>
